Pipeliner User Guide

Visual pipeline editor for orchestrating AI agent workflows.

Getting Started

Sign in with your Google account. From the home screen, you can create a new pipeline or open an existing one from your saved pipelines list.

add_circle

Create a Pipeline

Click New Pipeline, enter a name, and you will land on an empty canvas ready for agent nodes.

The Agent Palette

The left sidebar contains the agent palette — a categorized list of agents you can add to your pipeline:

I/O Agents

File Input, File Output, Google Drive, API Endpoint

Creative Agents

Copywriter, Art Director, Image Generator, Video Generator

Operational Agents

Strategist, Researcher, Quality Scorer, Compliance

Adding Nodes

Drag any agent from the palette onto the canvas to create a node. The node appears with input ports on the left and output ports on the right, ready to be connected.

You can also double-click an agent in the palette to add it to the center of the current viewport.

Connecting Nodes

There are two ways to wire nodes together:

drag_indicator

Method 1: Drag-to-Connect

1

Click and hold on an output port (right side of a node).

2

Drag the wire to an input port (left side of another node).

3

Release to create the connection. A colored wire appears between the ports.

ads_click

Method 2: Click-to-Connect

1

Double-click a port to arm it. The port shows a pulsing glow to indicate it is armed.

2

Compatible ports on other nodes light up to show valid targets.

3

Single-click any highlighted port to complete the wire.

?

Press Esc to cancel and disarm the port.

Node Configuration

Each node can be configured after placement:

  • Change agent type: Use the dropdown at the top of the node to switch to a different agent.
  • Delete a node: Click the X button in the node's top-right corner, or select the node and press Delete.
  • Edit parameters: Click the node body to open its configuration panel on the right side.

Port Types

Ports are typed to indicate what kind of data flows between agents:

  • string — Plain text (prompts, briefs, copy)
  • list — An ordered collection of items
  • dict — A key-value mapping (structured data)
  • number — Numeric values (scores, counts)
  • object — Complex objects (images, video references, documents)
Ports of incompatible types cannot be connected. The click-to-connect method makes compatible ports easy to identify — only valid targets light up.

Saving & Running

save

Save

Click the Save button in the toolbar to persist your pipeline to the database. Pipelines auto-save on significant changes, but manual saves ensure nothing is lost.

play_arrow

Run

Click Run to execute the pipeline. Nodes execute in topological order — each node waits for its inputs before firing. Progress is shown with animated borders on active nodes.

Import from Core Agents

When working in Core Agents, use the "Send to Pipeline" button to export a session's agent manifest. This opens Pipeliner with the pipeline pre-populated:

  • Agents appear as nodes on the canvas, already wired in execution order.
  • You can reconfigure, add, or remove nodes before running.
  • The imported manifest preserves agent parameters and context from the Core Agents session.

Canvas Navigation

  • Zoom: Scroll wheel to zoom in/out.
  • Pan: Click and drag on empty canvas space to pan.
  • Fit to screen: Press Ctrl/Cmd + 0 or use the fit button in the toolbar to center all nodes in view.
  • Select multiple: Hold Shift and drag to create a selection box.

File I/O

File nodes allow data to enter and leave the pipeline:

  • Upload files: Use a File Input node to upload local files (PDF, DOCX, images, video).
  • Google Drive import: Connect a Google Drive node and select files from your drive.
  • Export to Google Docs: Use a File Output node configured for Google Docs to write pipeline results as a formatted document.
